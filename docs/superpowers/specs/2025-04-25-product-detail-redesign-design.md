# 商品详情页重新设计

## 概述

全面重新设计商品详情页（`/products/:id`），采用经典奢华简约风格，对标 Cartier/Tiffany 的设计语言。核心变化：3D 模型从全屏抽屉改为嵌入图片区域，整体布局和视觉风格升级。

## 视觉风格

**经典奢华简约** — 与珠宝/奢侈品定位一致。

- **字体**: Cormorant Garamond（展示/标题）+ DM Sans（正文/UI），沿用现有字体体系
- **色彩**: 温暖中性色系 — `#1A1A1A`（主文字）、`#8A8986`（次文字）、`#D5D4D0`（边框）、`#F0EFED`（背景）、`#FAFAF8`（页面背景）
- **形状**: 直角边框（border-radius: 1-2px），拒绝圆角，体现传统奢侈品高级感
- **间距**: 宽松留白，信息层次分明

## 3D 模型嵌入交互

**核心变化：去掉全屏抽屉（FullscreenDrawer），3D 模型直接嵌入主图区域。**

### 交互流程

1. **图片模式（默认）** — 显示商品图片，缩略图条中 3D 项有绿色圆点标记
2. **点击 3D 缩略图** — 主图区域显示加载动画（旋转环 + "加载 3D 模型..."）
3. **加载完成** — `<model-viewer>` 直接渲染在主图区域，可旋转/缩放
4. **浮动控制栏** — 底部居中显示「自动旋转 ↻」和「全屏 ⤢」两个按钮
5. **退出 3D** — 点击任意图片缩略图，恢复图片模式

### 多 SKU 3D 模型切换

- 切换 SKU 时，如果新 SKU 有 `modelUrl`，自动加载对应 3D 模型
- 如果新 SKU 无 3D 模型，3D 缩略图隐藏

### 全屏模式

- 3D 浮动控制栏的「全屏」按钮仍触发全屏展示（使用 FullscreenDrawer）
- 全屏模式下显示 3D 模型 + SKU 缩略图切换（保留现有逻辑）

## 组件结构

重新组织为以下组件：

```
app/pages/products/[id]/index.vue          — 页面入口，数据获取，状态管理
app/components/product/
  ProductGallery.vue                        — 图片画廊（主图 + 缩略图 + 3D 集成）
  ProductInfo.vue                           — 右侧商品信息栏（桌面端 sticky）
  ProductSpecSelector.vue                   — 规格选择器（通用）
  ProductActions.vue                        — 操作按钮（加入购物车 / 立即购买 / 数量）
```

### 组件职责

- **ProductGallery**: 管理图片/3D 切换状态，包含主图、缩略图条、3D 模型渲染
- **ProductInfo**: 桌面端 sticky 信息栏，移动端滚动内容区，包含品牌标签、名称、描述、价格
- **ProductSpecSelector**: 通用的规格选择组件（材质、尺寸等），接收 specs 数据，emit 选择事件
- **ProductActions**: 数量选择 + 购物车/购买按钮，桌面和移动端共用

### 页面入口职责

- `useAsyncData` 获取商品数据
- SKU 选择状态管理
- 价格/库存计算逻辑
- 加入购物车 / 立即购买业务逻辑

## 数据流

```
[index.vue]
  ├── useAsyncData → getProductDetail(id)
  │     → { product, skus }
  ├── selectedSpecs: Record<string, string>   ← 用户选择
  ├── currentSku: computed                    ← 根据 specs 匹配 SKU
  ├── currentPrice: computed                  ← currentSku.price
  ├── currentStock: computed                  ← currentSku.stock
  ├── modelSkus: computed                     ← skus.filter(s => s.modelUrl)
  │
  ├── <ProductGallery>
  │     props: images, modelSkus, currentSku
  │     emit: 无（内部管理图片/3D 切换）
  │
  ├── <ProductInfo>
  │     props: product, currentPrice
  │
  ├── <ProductSpecSelector>
  │     props: skus, selectedSpecs
  │     emit: update:selectedSpecs
  │
  └── <ProductActions>
        props: currentSku, quantity
        emit: add-to-cart, buy-now, update:quantity
```

## 状态处理

沿用现有模式：
- **加载中**: 居中旋转加载动画
- **错误**: 错误提示 + 重试按钮
- **商品不存在**: "商品不存在" 空状态提示
- **SKU 缺货**: 库存为 0 时，购买按钮置灰并显示"缺货"

## 不在范围内

- 热门商品/新品推荐区域（保持现有 HotProducts/LatestProducts 不变）
- 面包屑导航（保持现有）
- Header/Footer（保持现有）
- 商品列表页（保持现有）
- 后端 API 变更
